# Xaver
This tool attempts to implement saving and loading as a tool for Exanima, and was inspired by the Salvus save manager tool by steam user Silk, which you can find at (https://steamcommunity.com/app/362490/discussions/0/4032473829604421294/).

## Features
Along with being able to make and restore backup saves for each save you have in the game, Xaver allows you to make several backups and name said backups to keep better track of them. Additionally, Xaver is able to display relevant information about the save, such as:
* Name of the character
* Level the character is on
* Time the save file was last modified
* Size of the file (Save sizes can range from a couple of Megabytes in the early game and around 26 Megabytes in the late game)
